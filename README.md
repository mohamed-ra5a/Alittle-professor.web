# Alittle-professor.web

Live Website:  
https://mohamed-ra5a.github.io/Alittle-professor.web/

## About
Alittle-professor.web is a simple educational website that helps new high school students understand different scientific fields in a clear and organized way. It explains what each subject studies and provides trusted resources for learning.

## Features
- Overview of major fields: Physics, Chemistry, Biology, Mathematics, Computer Science, Earth Science  
- Learning levels: Basic, Intermediate, Advanced  
- Resources page with recommended websites and YouTube channels  
- Responsive design for different devices
